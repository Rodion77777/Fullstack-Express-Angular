npm i nodemon --save-dev        установка зависимости позволяющая серверу отслеживать обновления на лету
--save-dev                      указываем зависимость как инструмент только для процесса разработки

npm install body-parser         фреймворк позволяющий парсить запросы и читать данные автоматически.
npm install cors morgan         два пакета устанавливаемых одной командой.
                                CORS позволяет отвечать клиентам которые сидят на другом домене (cors запросы).
                                MORGAN позволяет удобно логировать запросы. Логи будут видны в консоли.
npm i mongoose                  пакет который помогает работать с MongoDB

npm i bcrypt                    hash your password
npm i jsonwebtoken              token for user/session
npm install multer moment       moment работает с датами (для имен файлов), multer c загрузкой и проверкой файлов
