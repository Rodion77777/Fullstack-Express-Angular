Mongo DB

Characteristics:

- MQL: NoSQL база которая использует язык запросов MongoDB (MQL);
- реляционная база данных: набор данных с заданными взаимосвязями;
    (Реляционная модель объединяет данные в таблицы,
    где каждая строка представляет собой отдельную запись,
    а каждый столбец состоит из атрибутов, содержащих значения.)

- BSON Документоориентированная: хранит данные в виде JSON документов (или другие форматы),
    а затем организует их в коллекции;

- Хранит не структурированные данные: хорошо подходит для соц.сетей, СМИ, или Интернет вещей (IoT);
- ACID транзакции: атомарные, структурированные, последовательны, изолированы, долговечны;
- no SQL: для взаимодействия с базой данных не используются SQL команды;
